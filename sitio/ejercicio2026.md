# Día 5 - Proyecto final

## Actividad Final (Entrega: viernes)

Mini proyecto de análisis de datos utilizando un conjunto de datos proporcionado por la organización.

El proyecto debe incluir:

- Carga de datos.
- Exploración básica.
- Al menos una visualización.
- Una función propia.
- Conclusiones breves.

## Estructura recomendada

```text
README.md
datos/
figuras/
notebook.ipynb
```

| Actividad | Peso |

| Exploración de datos | 20% |

## Requisitos mínimos para participar en el hackatón

- Entregar las tres actividades.
- Tener al menos un commit en GitHub.
- Obtener una puntuación mínima de 60/100.
