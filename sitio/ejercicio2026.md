# Semana Pre-Hackatón

## Objetivo

Durante esta semana deberás familiarizarte con las herramientas que utilizaremos durante el hackatón:

- Git y GitHub
- JupyterHub y JupyterLab
- Python para análisis de datos
- R para análisis de datos
- Trabajo reproducible y colaborativo

La participación en el hackatón estará condicionada a la entrega de las actividades propuestas durante la semana.

# Día 5 - Proyecto final

## Actividad Final (Entrega: viernes)

Realizar un mini proyecto de análisis de datos utilizando un conjunto de datos proporcionado por la organización.

El proyecto debe incluir:

- Carga de datos.
- Exploración básica.

- Conclusiones breves.
